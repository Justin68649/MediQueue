 MediQueue - Healthcare Queue Management System

A complete, production-ready queue management system designed for healthcare facilities to reduce waiting times and improve patient experience.

 Features

 Patient Portal
- Join queue remotely from anywhere
- Real-time queue position tracking
- SMS and email notifications
- View appointment history
- Submit feedback and ratings

Staff Portal
- View department queue
- Call next patient
- Start and complete services
- Track service time metrics
- Performance analytics

 Admin Portal
- Complete system analytics dashboard
- User management (CRUD operations)
- Department management
- System configuration
- Report generation and export
- Audit logs viewer
- Database backup

 Public Display
- Real-time queue status display
- TV/LED screen support
- Audio announcements
- Auto-refresh functionality
